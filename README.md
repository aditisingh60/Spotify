
# 🎵 Spotify User Behavior Dashboard

## Problem Statement
Analyzed 50,000 Spotify users to identify key drivers of 
**churn**, **ad conversion**, and **engagement** — and built 
an interactive Tableau dashboard for product and marketing teams.

## Key Business Questions
| Pillar | Question |
|--------|----------|
| 🔴 Churn | Which countries and age groups churn the most? |
| 🟢 Ad Conversion | Which subscription plan converts best from ads? |
| 🔵 Engagement | Which genres and age groups are most engaged? |

## Stack
- Python (Pandas, NumPy) — data cleaning & feature engineering
- Tableau Public — dashboard & visualization

## How to Run
```bash
pip install -r requirements.txt
python scripts/clean.py     # generates data/cleaned/spotify_cleaned.csv
python scripts/analyze.py   # prints key insights in terminal
```
